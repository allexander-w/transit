Специальная таможенная процедура (СТП)

Возможность перемещения грузов без уплаты налогов при соблюдении условий СТП применяется для товаров, которые можно перемещать без стандартных пошлин при соблюдении специфических условий.

Наши специалисты помогут определить возможность применения СТП для вашего груза и подготовят необходимые документы.

Условия применения СТП:

  • Соответствие товара установленным требованиям
  • Наличие необходимой документации
  • Соблюдение целевого использования товара

ПЕРЕЧЕНЬ ВСЕХ ТАМОЖЕННЫХ ПРОЦЕДУР

Выпуск для внутреннего потребления

Включает введение импортируемых товаров в экономический оборот ЕАЭС после уплаты всех налогов и пошлин, позволяя использовать или продавать товары внутри Союза.

Экспорт

Управляет вывозом продукции за пределы ЕАЭС, требуя соблюдения всех необходимых критериев, включая уплату пошлин и соблюдение экспортных ограничений.

Таможенный транзит

Позволяет перевозить грузы через ЕАЭС без их использования, минимизируя задержки и освобождая от пошлин.

Временный ввоз (допуск)

Предоставляет возможность временного ввоза на территорию ЕАЭС с отсрочкой уплаты пошлин, подходящая для выставок и временных проектов.

Временный вывоз

Позволяет временно вывезти товары для определенных целей с возможностью последующего возврата и уплаты пошлин.

Переработка на таможенной территории

Обеспечивает возможность использования импортных материалов для производства, которое затем можно экспортировать без пошлин, стимулируя производство.

Переработка вне таможенной территории

Включает вывоз для переработки за рубежом и последующий ввоз готовой продукции, уменьшая таможенные расходы.

Переработка для внутреннего потребления

Требует переработки импорта для внутреннего рынка, с последующей уплатой всех пошлин перед выпуском в обращение.

Таможенный склад

Предоставляет место для хранения продукции под контролем до их окончательного использования, экспорта или уничтожения.

Свободный склад

Позволяет хранить продукцию без уплаты пошлин до их использования или реализации на внутреннем рынке.

Реэкспорт

Позволяет возвращать ввезенные товары без уплаты экспортных пошлин, полезно для временных проектов.

Реимпорт

Обеспечивает возврат грузов, временно вывезенных за границу, без дополнительных пошлин.

Уничтожение

Применяется к запрещенным к ввозу товарам, которые уничтожаются под контролем для соблюдения законов.

Отказ в пользу государства

Позволяет передать товары без коммерческой ценности или запрещенные к использованию в собственность государства для безопасности.

Специальная таможенная процедура

Применяется для товаров, которые можно перемещать без стандартных пошлин при соблюдении специфических условий.

Беспошлинная торговля

Обеспечивает продажу товаров в зонах duty-free, стимулируя туризм и потребительскую активность.

Как мы работаем?

Запрос и расчет
